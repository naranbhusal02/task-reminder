# ⏳ Smart Task Reminder  

**Task Reminder** is a web based application built for reality by teen to **increase producitivity** by providing feature like task reminders and **Journal** writing.


**Live Demo:** [taskreminder.naranbhusal02.xyz](https://taskreminder.naranbhusal02.xyz/)  

---

## ✨ Features  
- ⏲ **Custom Timer** – Set your own time as the countdown. 
- 🔔 **Loud Alarm** – Unignorable alarm sounds to remind you of the task
- 🔔 **Custom Alarm** – Add a custom alarm through youtube url or upload it from local storage as well.  
- 📝 **Journal** – Write and save your thoughts as journal on the application  
- 💬 **Popup Reminder** – Shows what task to be done after time is up with a popup  
- ⏳ **Snooze & Extend** – Lets you snooze the timer
- 📱 **Responsive UI** – Works seamlessly on desktop and mobile  
- 🎯 **Minimal Design** – Just what you need, nothing extra
